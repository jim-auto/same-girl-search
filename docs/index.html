Local-first face similarity search

画像から重複候補プロフィールを横断検索する OSS MVP

InsightFace で embedding を作成し、SQLite と FAISS に保存して、FastAPI と Next.js UI から top-k 類似候補を確認できます。

MVP Scope

まず実データで動く構成

Image Embedding

InsightFace と onnxruntime で顔検出、最大顔の normalized embedding、bbox、検出 score を生成します。

Similarity Search

FAISS の inner product index に正規化済み vector を保存し、cosine similarity として検索します。

Metadata Storage

SQLite に画像パス、source site、shop name、profile URL、bbox、embedding を保存します。

Duplicate Viewer

Next.js UI で画像をアップロードし、score と source metadata 付きの top-k 候補を確認します。

Quick Start

Docker でローカル実行

docker compose up --build docker compose run --rm api python scripts/index_folder.py --image-dir /workspace/data/images --site local --shop default
Web UI: localhost:3000 API docs: localhost:8000/docs

Responsible Use

lawful / ethical scraping を前提にする

  • Terms of Service と robots.txt を確認する
  • rate limit と User-Agent を必ず設定する
  • 個人情報の過剰保存を避け、削除しやすい構造にする
  • 検索結果は候補であり、同一人物と断定しない
